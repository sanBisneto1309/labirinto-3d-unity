# Labirinto 3D

Labirinto 3D desenvolvido em Unity 2022 como projeto de prática.

## Como jogar
- WASD ou setas para mover o jogador
- Chegue ao cubo verde para completar o labirinto

## Tecnologias
- Unity 2022.3.20f1
- C#

## Funcionalidades
- Movimentação do jogador
- Câmera seguindo o jogador
- Fim de labirinto com reinício
